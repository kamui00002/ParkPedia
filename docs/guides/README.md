# Guides and Setup Documentation

このディレクトリには、ParkPediaの各種セットアップガイドと実装手順書が含まれています。

## ファイル一覧

### セットアップガイド
- **APP_CHECK_SETUP_GUIDE.md** - Firebase App Checkのセットアップ手順
- **CRASHLYTICS_SETUP.md** - Firebase Crashlyticsの初期設定ガイド
- **CRASHLYTICS_BUILD_GUIDE.md** - Crashlyticsを含めたビルド手順
- **AD_DISPLAY_GUIDE.md** - AdMob広告の実装と表示ガイド

### 実装ガイド
- **QUICK_START_GUIDE.md** - プロジェクトのクイックスタートガイド
- **COMPLETE_IMPLEMENTATION_GUIDE.md** - 完全な実装手順書
- **TYPESCRIPT_MIGRATION_PLAN.md** - TypeScriptへの移行計画

## 使い方

新しい機能を実装する際や、開発環境をセットアップする際は、該当するガイドを参照してください。
